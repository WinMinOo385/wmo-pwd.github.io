# Professional Web Developer Framework

A curated collection of web development resources organized in an interactive, visual framework. Discover hosting services, learning platforms, coding challenges, tools, and more.

## UI/UX Design Credit

**This project's UI/UX design is inspired by and based on the [OSINT Framework](https://osintframework.com).**

Created by [Justin Nordine (@jnordine)](https://twitter.com/jnordine).  
- Website: https://osintframework.com  
- GitHub: https://github.com/lockfale/osint-framework

## Installation

```bash
npm install
npm start
```

Open `http://localhost:8000` in your browser.

## Usage

Browse the interactive framework visualization. Click categories to explore resources and visit links directly.

## Adding Resources

Edit `public/arf.json` to add new resources following the existing JSON structure.
